使用Let’s Encrypt生成通配符域名证书

安装cerbot：

apt install letsencrypt

生成通配符证书：

certbot certonly \
--manual \
--preferred-challenges=dns \
--email fzhujia@gmail.com \
--server https://acme-v02.api.letsencrypt.org/directory \
--agree-tos \
-d *.zhujia.dev
