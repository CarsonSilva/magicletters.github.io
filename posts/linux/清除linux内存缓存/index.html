清除Linux内存缓存，缓冲区和交换空间

如何清除缓存

每个Linux系统都有三个清除缓存的选项，而不会中断任何进程或服务。

  1. 只有清除页缓存

    # sync; echo 1 > /proc/sys/vm/drop_caches

  2. 清除dentries和inodes

    # sync; echo 2 > /proc/sys/vm/drop_caches

  3. 清除全部

    # sync; echo 3 > /proc/sys/vm/drop_caches

如何清除交换分区

如果要清除交换分区，执行以下命令：

# swapoff -a && swapon -a
